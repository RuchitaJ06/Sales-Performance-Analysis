# Sales Performance Analysis (2014–2017)

# Objective
Analyze sales data, identify trends, and create an interactive Power BI dashboard.

##  Tools Used
Python (Pandas, Matplotlib)
Power BI
Sklearn
Excel

## Key Analysis
- Total Revenue
- City Wise Sales
- Category Wise Sales
- New vs Returning Customer
- Monthly Sales Trend
- Future Sales Prediction (Linear Regression)

##  Key Insights
✔ Technology category highest revenue
✔ Monthly sales increasing trend
✔ Returning customers contribute more
✔ Top 20 cities generate max revenue
✔ Weak regions need offers & promotions
